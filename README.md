### Чек лист проверок для приложения https://www.saucedemo.com

1. Страница авторизации:
    - авторизация с валидными данными;
    - попытка авторизации с пустым значением в поле "User";
    - попытка авторизации с пустым значением в поле "Password";
    - попытка авторизации с невалидными значениями в полях "User" и "Password";  

*(проверки в Java Class name = "LoginTest)*
2. Страница с каталогом товаров:
    - проверка открытия карточки товара по его наименованию;

*(проверки в Java Class name = "ProductTest)*
3. Корзина товаров:
    - проверка возможности добавления товара в корзину;
    - проверка возможности удаления товара из корзины;

*(проверки в Java Class name = "CartTest)*
4. Страница учетных данных:
    - попытка сохранения валидных личных данных;
    - попытка сохранения личных данных с пустым значением в поле "First Name";
    - попытка сохранения личных данных с пустым значением в поле "Last Name";
    - попытка сохранения личных данных с пустым значением в поле "Zip Code";

*(проверки в Java Class name = "ChecloutTest)*
